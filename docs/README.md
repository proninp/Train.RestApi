# REST API in .NET

This repository is dedicated to learning and mastering **REST API development** in .NET.  
It serves as a personal playground for designing, building, and refining HTTP-based APIs following modern best practices.

## What’s inside

* Practical exercises covering REST fundamentals: resources, HTTP methods, status codes, and conventions  
* Example projects demonstrating clean API design, routing, and request/response handling  
* Work with validation, error handling, pagination, filtering, and versioning  
* Refactoring and improvement tasks focused on readability, maintainability, and consistency  
* Space for personal notes, experiments, and API design considerations

## Why this repository

* Gain a solid understanding of REST principles and HTTP semantics  
* Practice building clean, predictable, and consumer-friendly APIs  
* Learn to handle real-world API scenarios: validation, errors, evolution, and backward compatibility  
* Build a reusable foundation for future backend and integration projects
